# Titanic Survival Prediction

## Objective

Build a Machine Learning model to predict whether a passenger survived the Titanic disaster based on passenger information.

## Dataset

The dataset contains passenger details such as:

* Passenger Class (Pclass)
* Gender (Sex)
* Age
* Fare
* Number of Siblings/Spouses Aboard (SibSp)
* Number of Parents/Children Aboard (Parch)
* Embarked Location

Target Variable:

* Survived

  * 0 = Did Not Survive
  * 1 = Survived

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Cleaning
4. Feature Engineering
5. Train-Test Split
6. Model Training using Random Forest Classifier
7. Prediction and Evaluation

## Results

* Model: Random Forest Classifier
* Accuracy: Approximately 82%

## Conclusion

The model successfully predicts passenger survival with approximately 82% accuracy. Features such as passenger class, gender, age, fare, and embarkation location were used to determine survival probability.
